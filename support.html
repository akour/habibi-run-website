Basic troubleshooting

  1. Close and reopen the game.
  2. Restart the device.
  3. Confirm the device has enough free storage.
  4. Install the latest available test build only from an authorized source.
  5. Do not clear app data unless you accept that local progress may be removed.

Report a bug

Please include the following:

  • Device model
  • Android version
  • Game version
  • Clear description of the issue
  • Steps that reproduce it
  • Screenshot or screen recording, when available
