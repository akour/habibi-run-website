Privacy Policy

App: Juha’s Bad Day: Habibi, Run!
Developer: One Apps
Package ID: com.oneapps.habibirun
Effective date: July 4, 2026

Important: This policy describes the currently planned no-tracking build. Final behavior must be verified against the release APK/AAB, Android permissions, network traffic, plugins, and SDKs before launch.

Information collection

The current design does not require an account or login. One Apps currently intends the app to operate without advertising, analytics, location collection, contacts access, camera access, microphone access, or collection of personal information. These statements remain provisional until the release build has been technically verified.

Local save data

The app is expected to store game progress and settings locally on the user’s device. Local save data remains on the device unless the user deletes the app data, uninstalls the app, restores the device, or a future feature changes this behavior.

Permissions

No sensitive permissions are currently planned. The final Android manifest and any permissions added by Godot, plugins, or third-party SDKs must be inspected before release.

Third-party services

No advertising, analytics, crash-reporting, authentication, cloud-save, payment, social, or push-notification service is currently planned. If any third-party service is introduced, this policy will be updated before release or before the feature becomes available.

Children’s privacy

The game is not currently intended specifically for children. One Apps does not knowingly design the current build to collect personal information from children. The target-audience declaration will be reviewed before publication.

Data security

Because the planned build stores only local game data, users should protect access to their devices. One Apps will review any future networked systems and service providers before introducing them.

Data retention

Local save data is retained on the device until the user clears app data, uninstalls the app, or the operating system removes it. No server-side retention is currently planned.

Sale or sharing of personal data

The current design does not include selling or sharing personal data. This remains subject to verification of the final binary and all included services.

Changes to this policy

This policy may be updated as development continues. The effective date above will be revised when material changes are made.
